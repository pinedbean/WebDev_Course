Full-Stack Web Dev · Module 1 — HTML Foundations

Chunk 1.1 — Lab: Build Your "About Me" Page

🧪 ASSIGNMENT ⏱️ 45–75 min

🎯 Your mission

Create a single web page, about.html, that introduces you. You'll use headings, paragraphs, links, and an image — and open it in your browser to see it render. This page becomes part of your personal site that grows across Module 1.

Before you start

  • Make a folder for this project, e.g. ~/Desktop/webdev-course/module-01-html/personal-site/.
  • Open it in VS Code (code .).
  • Have one image ready (a photo, an avatar, or any .jpg/.png) saved in that folder. No image handy? Use a placeholder URL like https://placehold.co/300x300.
⚠️ Try it yourself first
Build as much as you can from memory and the lecture. Only open solution.html when you're stuck or to compare at the end.

Tasks

1 Create the file & skeleton

Create about.html and type out the full HTML skeleton (DOCTYPE, html, head with charset + viewport + title, body). Set the <title> to your name.

💡 Shortcut
In VS Code, type ! then press Tab in an HTML file — it expands a full skeleton for you (Emmet). Try it, then read what it generated.

2 Add a main heading

Inside the <body>, add a single <h1> with your name.

3 Add an intro & sections

Add an <h2>About Me</h2> followed by 1–2 <p> paragraphs about yourself. Then add a second <h2> (e.g. "What I'm Learning") with another paragraph.

4 Add your photo

Add an <img> with a meaningful alt describing the picture.

5 Add at least two links

Add a paragraph with links — for example, an external link to something you like (opens a website) and a mailto: link to email you.

6 Open it in the browser

Double-click about.html (or right-click → Open With → your browser). Confirm everything renders: heading, text, image, and clickable links.

✅ Deliverable — acceptance checklist

  • about.html exists and opens in a browser.
  • It has a complete, valid skeleton (DOCTYPE, html lang, head, body).
  • Exactly one <h1>, plus at least two <h2> sections.
  • At least two paragraphs of real text.
  • One <img> with descriptive alt text that actually displays.
  • At least two working <a> links (one external, one mailto:).

🚀 Stretch goals (optional)

  • Validate your page at validator.w3.org (paste your code). Fix any errors.
  • Add a horizontal rule <hr> between sections and a bold/italic word with <strong>/<em>.
  • Add a link that opens in a new tab using target="_blank".
  • Commit your work: git add . && git commit -m "Add about page".
